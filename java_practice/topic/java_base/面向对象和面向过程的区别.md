### 40.面向对象和面向过程的区别？
>面向过程就是分析出解决问题所需要的步骤，然后用函数把这些步骤一步一步实现，使用的时候个依次调用就可以了。面向过程就好像一个细心的管家，事无巨细的都要考虑到。             
 面向对象是把构成问题的事物分解成各个对象，建立对象的目的不是为了完成一个步骤，而是为了描叙某个事物在整个解决问题的步骤中的行为。
>面向对象就好像一个家用电器，不用什么都考虑到，只需要知道它的功能，不需要知道它的工作原理。                                          
 以汽车出站和汽车到站为例：              
 对于面向过程来说，这是两个事件，汽车出站是一个事件，启程到站是一个事件，面向过程编程的过程中关心的是事件，而不是汽车本身。针对这两个事件，编码两个函数，然后依次调用即可。              
 对于面向对象来说，关心的是汽车这类对象，两个事件只是这类对象所具有的行为，而且对于这两个行为的顺序没有强制的要求。
