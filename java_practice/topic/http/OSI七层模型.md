### 6.OSI的七层模型都有哪些？
>应用层：网络服务与最终用户的一个接口。HTTP，HTTPS，FTP，POP3、SMT                    
 表示层：数据的表示、安全、压缩。                       
 会话层：建立、管理、终止会话。                    
 传输层：定义传输数据的协议端口号，以及流控和差错校验。 TCP UDP               
 网络层：进行逻辑地址寻址，实现不同网络之间的路径选择。  IP ICMP RIP IGMP                      
 数据链路层：建立逻辑连接、进行硬件地址寻址、差错校验等功能。    ARP                 
 物理层：建立、维护、断开物理连接。
